# GitHub Setup Instructions

## Repository Description

Use this description for your GitHub repository:

```
Interactive typographic grid calculator for designing pixel fonts and grid-based posters. 
Paint custom font layouts with clickable cells, adjust grid sizes, and export printable designs. 
Built with pure HTML/CSS/JavaScript.
```

## Steps to Complete Setup

### 1. Create Repository on GitHub
- Go to https://github.com/new
- Repository name: `grid-calc`
- Description: (use the text above)
- Visibility: Public
- Click "Create repository"

### 2. Push Your Local Repository
Run these commands in your project directory:

```bash
git remote add origin https://github.com/YOUR_USERNAME/grid-calc.git
git branch -m main
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to your repository on GitHub
2. Click "Settings" tab
3. In the left sidebar, click "Pages" (under "Code and automation")
4. Under "Source", select:
   - **Branch:** main
   - **Folder:** / (root)
5. Click "Save"
6. GitHub will build and deploy your site
7. After a few moments, you'll see the live URL at the top of the Pages section

### 4. Access Your Site
Your hosted tool will be available at:
```
https://YOUR_USERNAME.github.io/grid-calc/
```

Simply visit this URL to use the calculator from anywhere!
